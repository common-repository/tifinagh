== Description ==

This plugin allows you to create your website in Tifinagh with various fonts and lets you upload your own font files and apply them to any element of your website without requiring
a knowledge of html or css. All .otf and .ttf font files are supported.

Forget about using Flash and embed your fonts the right way.

Font Uploader embeds the fonts directly onto your web page using @font-face via CSS.

== Installation ==

1. Upload the 'tifinagh' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Tifinagh and configure your fonts

== Frequently Asked Questions ==

- Can I use this plugin to apply tifinagh fonts to any html element, or just a select few? -

You can apply tifinagh fonts to any element you wish using the "Advanced Custom Elements".
